# Ntuaflix

**Full-Stack Movie & TV Series Database Platform** 

*Developed for the Software Engineering Course (2023-2024) @ ECE NTUA*

## Prerequisites
- Node.js
- Yarn (Package Manager for JavaScript)
- MySQL Server
- Sequrity: SSL certificates

## Setup
1. Clone the repository: `git clone https://github.com/...`
2. Change into the project directory: `cd softeng23-07`
3. Create a database named softeng and import the schema `ddl.sql`
4. Install dependencies using `yarn install` in the main directories:
   - front-end
   - back-end
   - cli-client
5. Running the project:
   - Ntuaflix API: `localhost:9876`
   - Front-end application: `localhost:3000`
