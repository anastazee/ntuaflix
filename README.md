# softeng23-07 - Ntuaflix
This is our project's repo for the Software Engineering course 2023-2024, ECE NTUA.
Ntuaflix API runs on a web server on localhost:9876. The front-end application is available on localhost:3000.

## Prerequisites
- Node.js
- Yarn (Package Manager for JavaScript)
- mysql softeng database using the repo's ddl
- ssl certificates used for https

## Setup
1. Clone the repository: `git clone https://github.com/...`
2. Change into the project directory: `cd softeng23-07`
3. Install dependencies using Yarn: `yarn install` in the main directories:
   - front-end
   - back-end
   - cli-client
